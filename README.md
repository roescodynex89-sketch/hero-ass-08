# SunCart – Summer Essentials Store ☀️

SunCart is a modern eCommerce platform for seasonal summer products. Users can browse products, view details, and manage their profiles with secure authentication.



## 🌟 Key Features
- **Authentication:** Secure Login and Registration using **BetterAuth**.
- **Social Login:** Quick access via **Google Login**.
- **Protected Routes:** Product Details page is only accessible to logged-in users.
- **Profile Management:** View and update user profile (Name and Photo).
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
- **Global Loading:** Smooth UI transitions using **React Icons**.

## 🛠️ Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Authentication:** BetterAuth
- **Icons:** React Icons
- **Database:** MongoDB
- **Hosting:** Vercel

## 📦 Main NPM Packages Used
- `better-auth`
- `react-icons`
- `mongodb`
- `tailwindcss`

## ⚙️ Environment Variables
To run this project, add the following variables to your `.env.local` file:
- `BETTER_AUTH_SECRET`
- `BETTER_AUTH_URL`
- `MONGO_URI`
- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`
